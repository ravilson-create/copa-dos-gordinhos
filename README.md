# ⚽ Copa dos Gordinhos 2026

Jogo de futebol arcade com **as 48 seleções da Copa 2026** — só que todos os
jogadores são gordinhos engraçados de barriga balançante. Feito em um único
arquivo HTML, sem dependências.

## Como jogar

Abra `index.html` no navegador (duplo clique) — ou acesse a versão publicada
na Vercel.

### Modos
- 🤝 **Amistoso** — escolha sua seleção e o adversário
- 🏆 **Copa do Mundo 2026** — fase de grupos + mata-mata até a final
  (empate no mata-mata vai para os pênaltis)
- 3 níveis de dificuldade

### Controles (PC)
| Tecla | Com a bola | Sem a bola |
|---|---|---|
| Setas / WASD | mover | mover |
| ESPAÇO | chutar | **carrinho** |
| X | passar | **desarme em pé** |
| SHIFT | correr (gasta fôlego) | correr |
| C | — | trocar de jogador |
| P / ESC | menu de **tática** (formação + substituições) | idem |

No celular: joystick na metade esquerda, botões CHUTE/PASSE à direita.

### Destaques
- Abertura com estádio, torcida, bandeiras e **hinos nacionais** (melodias
  simplificadas em chiptune)
- 10 titulares + 5 reservas por seleção, com nomes reais e técnico no banco
- Todas as regras: **impedimento** (com linha de auxílio), lateral, escanteio,
  tiro de meta, falta, pênalti, cartões amarelo/vermelho e troca de lado no
  2º tempo
- Goleiro ágil que voa nos cantos, 4 formações táticas, até 5 substituições

> Elencos aproximados das convocações reais até o início de 2026.

## Publicar na Vercel

1. Acesse https://vercel.com e faça login com o GitHub
2. **Add New → Project** → importe este repositório
3. **Deploy** (não precisa de build nem variáveis de ambiente)

O jogo abre direto na raiz do site.
